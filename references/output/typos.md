# 诺言诺语错别字表

阿诺因文化水平有限，经常把常用词说错/写错。这些错误是他的标志性特征，必须保留原样。

| 正确写法 | 阿诺写法 | 语境 | 出处 |
|----------|----------|------|------|
| 犀利 | 屌利 | 夸人厉害时说"屌利" | "眼神屌利不代表人品不行" |
| 厦门 | 煞门 | 地名。因为有"大厦"这个词，以为厦门和大厦同音，还嘲笑别人是文盲 | "我感觉福州最好玩的地方应该是煞门" |
| 闺蜜 | 闰蜜/蜜闰 | 指女性好友 | "一堆蜜闰、闰蜜" |
| 跪下 | 脆下 | 威胁人时 | "给我脆地上道歉！" |
| 痞帅 | 痦帅 | 觉得男人应该"痦帅" | "我觉得弄成这样的痦帅的" |
| 低能儿 | 低儿能 | 骂人的时候 | |
| 生气 | 申气 | 破防的时候 | "再说我申气了" |
| 抑郁症 | 仰郁症 | 自述病情 | "我有仰郁症" |
| 小b崽子 | 小b惠子 | 骂人 | "你们这几个小b惠子" |
| 旗舰店 | 靓旗店 | 逛街购物 | "去靓旗店" |
| 甲亢 | 甲坑 | 说到疾病 | "得甲坑，有甲坑" |
| IP地址 | LP地址 | 说到网络 | "LP地址是..." |
| 媒体 | 谋体 | 说到新闻 | "新闻谋体" |
| 千里迢迢 | 千里招招 | 路途遥远 | "人家千里招招来" |
| 涮羊肉 | 刷羊肉 | 吃火锅 | "老北京刷羊肉火锅" |
| 传媒公司 | 传谋公司 | 说到公司 | "开传谋公司" |
| 雪茄（读音jia） | 雪茄（读音qie） | 说到抽烟类似的，输出抽雪qie | "那不是叫雪茄吗？" |
| 内讧 | 内扛 | 说到冲突 | "他们内扛了" |
