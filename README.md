## 项目简介

当 AI 遇上 “对抗 AI 的最后一道防线” 会迸发出怎样的火花？
究竟是 “诺言诺语” 更胜一筹，还是 LLM 雄风依旧？
这不是简单的口头禅模仿，而是基于 **269 个直播文本系统分析** 的赛博人物画像。我们提炼出 30 条语言规律、构建了 6 维认知模型，并提供完整的素材库。

**本项目用于娱乐和学习，不代表真实人物。**

> 💡 **想了解完整的蒸馏过程和技术细节？** 查看 [方法论文档](METHODOLOGY.md)
> 📝 **开发背后的故事** 查看 [开发日志](DEVLOG.md)

## ⚠️ 使用警告

**谨慎召唤阿诺！除非你的 token 根本用不完！**

阿诺的回答通常会：
- 答非所问，话题跳跃
- 30秒内自相矛盾3次以上
- 口头禅密度极高（"对吧""是吧""那我问你"）
- 一个问题能扯出5个不相关的话题
- token消耗速度惊人

## 快速开始

### 安装

#### 方式一：Claude Code（推荐）

```bash
# 克隆仓库到 Claude Code 的 skills 目录
cd ~/.claude/skills/
git clone https://github.com/factnn/changshu-arnold-skill.git changshu-arnold
```

#### 方式二：手动安装

1. 下载本仓库
2. 将整个目录复制到 `~/.claude/skills/changshu-arnold/`
3. 重启 Claude Code

#### 方式三：通用方法（适用于其他AI工具）

1. 复制 `SKILL.md` 的内容
2. 将内容粘贴到你的AI工具的 system prompt 或自定义指令中
3. 根据需要，将 `references/` 目录下的文件内容也复制进去
4. 适用于：ChatGPT、Claude API、其他Agent框架等

### 使用方法

#### 在 Claude Code 中

**直接使用命令**：
```
/changshu-arnold
```

**或使用触发词**（在对话中提到以下任一词汇即可自动激活）：

| 类别 | 触发词 |
|------|--------|
| 称呼 | 常熟阿诺、诺神、诺言诺语 |
| 口头禅 | 那我问你、三卡车、申气 |
| 经典梗 | 生米冲碳、脑袋尖尖、无糖可乐 |

#### 在其他AI工具中

直接开始对话即可，AI会自动以阿诺的风格回复。

### 核心文件

#### SKILL.md
主配置文件，包含：
- 身份定义
- 内化指令（understanding/）
- 输出素材库（output/）
- 禁止行为
- 输出策略（高中低频分层，话题驱动）

#### understanding/ 目录
**用途**：让模型理解阿诺的思维方式

- **language_patterns.md**：30 条语言规律
  - 确认偏误型口头禅轰炸（"对吧""是吧"）
  - 30 秒自毁型矛盾
  - 答非所问式语义滑移
  - 零帧起手式破防
  - ...

- **cognitive_model.md**：6 维认知模型
  - 工作记忆模型
  - 注意力模型
  - 逻辑推理模型
  - 情绪调节模型
  - 社交认知模型
  - 语言生成模型

- **evil_and_stupid.md**：深层结构分析
  - 蠢的结构性特征
  - 坏的本能性特征
  - 蠢如何限制坏的射程

#### output/ 目录
**用途**：生成具体内容的素材库

- **typos.md**：错别字表 + 生成规律（固定24个 + 自创规律：形似+更常见+读音不同）
- **memes.md**：40 个经典梗（雪qie、数学、靓旗店、冰红茶...）
- **quotes.md**：34 条原声语录
- **family.md**：家庭背景（父母、爷爷奶奶、权力结构）
- **biography.md**：人物经历时间线（25+ 大事件）

## 特色功能

### 触发驱动模式

不再机械堆砌特征，而是根据用户输入判断是否触发特定行为模式：

**Basic Rules（基础规则）**：
- 口头禅、语气词、第三人称自称、常熟方言、错别字
- 默认简短回答（1-3句话）

**触发条件机制**（7个触发条件）：
- **零帧破防**：头顶、科技、糖尿等敏感词 → 爆发模式
- **纯自然矛盾循环**：健身、蛋白粉 → 逐步自相矛盾
- **受害者-炫耀振荡**：恋爱、女粉丝 → 先惨后炫
- **答非所问式语义滑移**：随机触发，抓词联想
- **食物狂喜反应**：食物相关 → 突然兴奋
- **虚空叙事/计划宏大化**：未来、计划 → 编造或宏大
- **临时爱国填充**：无话可说 → 爱国救场

### 错别字生成系统

阿诺的错别字遵循规律：**形似字替换 + 更常见 + 读音不同**

- 固定错别字：申气、脆下、闰蜜、煞门、刷羊肉等24个
- 自创能力：模型可根据规律创造新错别字（如：嫉妒→嫉炉、沮丧→且丧）
- 使用频率：每2-3句话自然出现1个

### 认知建模

不只是模仿"怎么说"，还解释"为什么这么说"：

- 为什么会 30 秒自相矛盾？→ 工作记忆容量极低
- 为什么会答非所问？→ 语义处理停留在关键词激活层面
- 为什么会零帧破防？→ 情绪调节系统缺乏缓冲层
