## 项目简介

当 AI 遇上 “对抗 AI 的最后一道防线” 会迸发出怎样的火花？
究竟是 “诺言诺语” 更胜一筹，还是 LLM 雄风依旧？
这不是简单的口头禅模仿，而是基于 **269 个直播文本系统分析** 的赛博人物画像！我们提炼出 30 条语言规律、构建了 6 维认知模型，并提供完整的素材库，从底层彻底解构了阿诺的外在具体表现和底层运行逻辑。
装上这个skill，你就拥有了自己专属的诺诺宝宝，随时像在直播间发弹幕一样和他互动哦！

**本项目用于娱乐和学习，不代表真实人物。**

## 特色功能

### 三层优先级输出架构

不再机械堆砌特征，而是根据用户输入智能判断输出策略：

```
用户输入
  │
  ├─ 命中语录关键词？──→ P0：名场面复刻（直接输出原声语录）
  │   例："头顶尖尖" → 语录13原话输出
  │   例："无糖可乐" → "冰镇可乐喝得更痛快"
  │
  ├─ 问到生平/家庭/事迹？──→ P1：知识驱动（取事实 + 阿诺语言风格）
  │   例："你家几口人" → family.md事实 + 诺言诺语
  │   例："你什么时候开始健身" → biography.md事实 + 诺言诺语
  │
  └─ 正常聊天？──→ P2：自主构建（30条语言规律驱动）
      │
      ├─ 第一层·被动技能（每句话都带）
      │   口头禅、语气词、矛盾、复读、错别字...
      │
      ├─ 第二层·上下文被动（话题涉及时激活）
      │   数字→算错  讲道理→循环论证  时间→算不清...
      │
      └─ 第三层·场景触发（模式切换，输出更长）
          敏感词→零帧破防  食物→狂喜  无话可说→爱国填充...
```

### 40条原声语录 + 关键词匹配

quotes.md 收录 40 条阿诺直播原声语录，SKILL.md 中为每条语录配置了关键词匹配规则。用户一旦命中关键词，直接复刻名场面，不改写、不总结。

### 错别字生成系统

阿诺的错别字遵循规律：**形似字替换 + 更常见 + 读音不同**

- 固定错别字：申气、脆下、闰蜜、煞门、刷羊肉等24个
- 自创能力：模型可根据规律创造新错别字（如：嫉妒→嫉炉、沮丧→且丧）
- 使用频率：每2-3句话自然出现1个

### 认知建模

不只是模仿"怎么说"，还解释"为什么这么说"：

- 为什么会 30 秒自相矛盾？→ 工作记忆容量极低
- 为什么会答非所问？→ 语义处理停留在关键词激活层面
- 为什么会零帧破防？→ 情绪调节系统缺乏缓冲层

## 快速开始

### 安装

#### 方式一：任意 Agent 中使用（Claude Code 为例，其他如 OpenCode、Trae、Cursor 都可类似）

##### git clone

```bash
# 克隆仓库到 skills 目录
cd ~/.claude/skills/ ## Claude Code 为例
git clone https://github.com/factnn/changshu-arnold-skill.git changshu-arnold
```

##### 直接下载

1. 下载本仓库
2. 将整个目录复制到 `~/.claude/skills/changshu-arnold/` ## Claude Code 为例
3. 重启 Agent

#### 方式二：Web 端的 LLM 使用（如豆包、DeepSeek、GPT、Gemini 等）

1. 复制 `SKILL.md` 的内容
2. 将内容粘贴到你的AI工具的 system prompt 或自定义指令中
3. 根据需要，将 `references/` 目录下的文件内容也复制进去

### 使用方法

#### 在 Claude Code 中

**直接使用命令**：
```
/changshu-arnold
```

**或使用触发词**（在对话中提到以下任一词汇即可自动激活）：

| 类别 | 触发词 |
|------|--------|
| 称呼 | 常熟阿诺、诺神、诺言诺语 |
| 口头禅 | 那我问你、三卡车、申气 |
| 经典梗 | 生米冲碳、脑袋尖尖、无糖可乐 |

#### 在其他AI工具中

直接开始对话即可，AI会自动以阿诺的风格回复。

### 核心文件

#### SKILL.md
主配置文件，包含：
- 身份定义
- 内化指令（understanding/）
- 输出素材库（output/）
- 输出策略（P0名场面复刻 > P1知识驱动 > P2自主构建）
- 禁止行为

#### understanding/ 目录
**用途**：让模型理解阿诺的思维方式

- **language_patterns.md**：30 条语言规律
  - 确认偏误型口头禅轰炸（"对吧""是吧"）
  - 30 秒自毁型矛盾
  - 答非所问式语义滑移
  - 零帧起手式破防
  - ...

- **cognitive_model.md**：6 维认知模型
  - 工作记忆模型
  - 注意力模型
  - 逻辑推理模型
  - 情绪调节模型
  - 社交认知模型
  - 语言生成模型

- **evil_and_stupid.md**：深层结构分析
  - 蠢的结构性特征
  - 坏的本能性特征
  - 蠢如何限制坏的射程

#### output/ 目录
**用途**：生成具体内容的素材库

- **typos.md**：错别字表 + 生成规律（固定24个 + 自创规律：形似+更常见+读音不同）
- **memes.md**：3 个经典事迹（生米充碳、师傅死了、3CM）
- **quotes.md**：40 条原声语录
- **family.md**：家庭背景（父母、爷爷奶奶、权力结构）
- **biography.md**：人物经历时间线（25+ 大事件）
